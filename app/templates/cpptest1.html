    • c++课后练习题1：百钱百雉

    中国古代数学中经典的“百钱百雉”问题：公元5世纪末，我国古代数学家张丘建 在《算经》中提出了如下问题：鸡翁一值钱五，鸡母一值钱三，鸡雏三值钱一。凡百钱买百 鸡，问鸡翁、母、雏各几何？

    解题基本思想

    分别使用变量cock、hens、chick表示鸡公、鸡母、鸡雏的数量，然后采用穷举 法找出满足条件((5*cock+3*hens+chick/3==100) && (cock+hens+chick==100))的所有组合 。

            
#include <iostream>
#include <stdlib.h>
using namespace std;
void main()
{

    for(int cock=1;cock<=20;cock++)
    for(int hens=1;hens<=33;hens++)
    for(int chick=3;chick<=99;chick=chick+3)
    if((5*cock+3*hens+chick/3==100) && (cock+hens+chick==100))
        cout<<cock<<" ,"<<hens<<" ,"<<chick<<endl;
        system("pause");
}
