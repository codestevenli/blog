    • 数据类型的可变与不可变

    • 不可变类型：int,string,tuple
    • 可变类型：list,tuple,dict
            

>>> a = "haha"
>>> id(a)
140507461886192
>>> a = "hahaha"
>>> id(a)
140507461886144
>>>
>>> b = [1,2,3]
>>> id(b)
140507461748712
>>> b.append(4)
>>> b
[1, 2, 3, 4]
>>> id(b)
140507461748712
>>>
        
        
    由上面可以看出，字符串string是不可变型，因为改变了字符串的字符其唯一辨识符也相应发生变化；列表list是可变型，因为列表中增加了一个数字4，但其唯一辨识符没有发生
改变。
